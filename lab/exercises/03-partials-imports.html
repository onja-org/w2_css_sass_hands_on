Your Mission: Organize This Code!

This page has many different components. Instead of writing all the CSS in one file, you'll split it into organized partials:

  1. Create a partials folder: scss/exercises/partials/
  2. Split your code into logical files:
    • _variables.scss - Colors, fonts, spacing
    • _mixins.scss - Reusable mixins and functions
    • _base.scss - Reset, typography, basic styles
    • _buttons.scss - All button styles
    • _navigation.scss - Navigation bar
    • _hero.scss - Hero section
    • _categories.scss - Category cards
    • _newsletter.scss - Newsletter section
    • _footer.scss - Footer styles
  3. Import everything: Use @import in your main file
  4. See the magic: Same result, organized code!
